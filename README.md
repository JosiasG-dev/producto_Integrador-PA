# producto_Integrador-PA

Creacion de un punto de venta con flujo MVC de una tienda de abarrotes, utilizando Java swing, phyton y C# NET MAUI
El proyecto de dividio en varias ramas dentro del repositorio para una mejor organizacion, siendo la rama MASTER donde se elojo la version principal hecha en lenguaje java
La rama Phyton que contiene el proyecto en ese lenguaje y finalmente la rama C# que contiene el proyecto descrito en ese lenguaje de programacion
